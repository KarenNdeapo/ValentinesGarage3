# Valentine Garage App

## MAP711S Assignment

This mobile application was developed for Valentine’s Garage.
The system allows truck check-in for repairs and maintenance.
It helps mechanics track tasks completed on vehicles and allows the owner to view reports.

---

# Features

## Truck Check-In
- Capture truck number
- Capture driver name
- Capture kilometers
- Capture vehicle condition

## Mechanic Tasks
- Mechanics can view task screen
- Notes can be added for vehicles

## Reports
- View repair reports
- View vehicle condition reports
- Track employee work completed

---

# App Architecture

The application uses a layered architecture.

## UI Layer
Handles screens and layouts.

## Data Layer
Handles truck information and storage.

## Repository Layer
Connects the UI and database.

1. Open Android Studio
2. Open the ValentineGarage project
3. Allow Gradle to sync
4. Click Run ▶
5. Select emulator or Android device
