# UI Changes Made

The project functionality was not changed. The update focused on making the app look original for the assignment.

## Branding

- Changed visible app identity to **EG GarageFlow**.
- Changed Android app label to **EG GarageFlow**.
- Replaced old logo with a new custom vector logo.
- Updated login title, report title, and report footer text.

## Visual design

- Replaced the previous orange/steel look with a teal and navy workshop theme.
- Rounded cards, buttons, text fields, dialogs, chips, and floating action buttons.
- Updated primary button gradient to navy-to-teal.
- Updated screen background and card colours.
- Updated top bars, active tab colours, progress indicators, and status styling.

## Screens affected

- Login screen
- Mechanic repair queue
- Vehicle job intake form
- Service detail/task screen
- Admin overview dashboard
- Service timeline/report screen
- Profile screen

## What was not changed

- Firebase Authentication
- Firestore repositories
- Role-based navigation logic
- ViewModels
- Data models
- Task/check-in workflow
- Admin and mechanic functionality
